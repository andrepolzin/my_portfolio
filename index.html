Quiz Game

This project is an Interactive Quiz Game that was created based on technologies such as: vanilla JavaScript, CSS, HTML, LocalStorage, APIs. The quiz game is based on multiple-choice questions along with a timer. Then, the answers are saved on Local Storage, being printed out on the screen by the end of the game.

To do List

To do list is a project that allows people to create a list of tasks that can be deleted or edited if needed. It was developed on JavaScript, HTML, CSS and everytime the 'Save task' button is pressed, the information is stored on the LocalStorage and then shown on the screen.

About
